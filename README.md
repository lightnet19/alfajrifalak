# Alfajri

Sistem Rukyat Digital berbasis Ilmu Falak.

## Struktur
- `public/` = semua file yang dideploy ke Vercel
- `public/core/` = engine falak
- `public/js/` = UI dashboard
- `public/dashboard/` = halaman dashboard
- `public/index.html` = landing page

## Jalankan lokal
```bash
npm install
npm run dev
```

Buka alamat yang muncul di terminal.

## Deploy ke Vercel
1. Push project ke GitHub.
2. Buka Vercel.
3. Import repository.
4. Deploy.

## Catatan
Kalkulasi falak di project ini sudah disusun agar siap dipakai sebagai web app produksi, namun untuk kebutuhan ilmiah resmi tetap perlu validasi lapangan dan kalibrasi metode sesuai standar lembaga yang digunakan.
